# 📥 FB Group Contact Collector

תוסף כרום לאיסוף פרטי מבקשי הצטרפות לקבוצות פייסבוק — שמות, אימיילים ומספרי טלפון.

## התקנה

1. פתח את Chrome ועבור לכתובת: `chrome://extensions/`
2. הפעל **מצב מפתח** (Developer Mode) בפינה הימנית העליונה
3. לחץ על **"טען תוסף ללא אריזה"** (Load unpacked)
4. בחר את התיקייה `fb-group-collector`
5. התוסף מותקן! ✅

## שימוש

1. היכנס לקבוצת פייסבוק שאתה מנהל שלה
2. עבור לדף **בקשות חברות** (Member Requests)
3. יופיע כפתור כחול צף **"📥 איסוף פרטים"** בפינה השמאלית התחתונה
4. לחץ על הכפתור — התוסף יגלול אוטומטית לטעון את כל הבקשות ואז יאסוף את הפרטים
5. בחלון התוצאות:
   - **📥 הורד CSV** — הורדת קובץ עם כל הפרטים
   - **📋 העתק ללוח** — העתקה מהירה להדבקה בגיליון אלקטרוני

## עמודות בקובץ CSV

| עמודה | תיאור |
|-------|--------|
| שם | שם מלא של המבקש |
| אימייל | כתובת אימייל (מתוך תשובות לשאלות הקבוצה) |
| טלפון | מספר טלפון (מתוך תשובות לשאלות הקבוצה) |
| תאריך | תאריך האיסוף |

## הערות

- התוסף עובד רק בדף בקשות חברות של קבוצה שאתה מנהל בה
- פייסבוק משנה את מבנה הדף מעת לעת — אם התוסף מפסיק לעבוד, ייתכן שצריך לעדכן את הסלקטורים
- הנתונים לא נשמרים ולא נשלחים לשום מקום — הכל מקומי בדפדפן שלך
